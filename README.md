# Operation NCLC 7

6-month French learning plan to achieve NCLC 7 (CEFR B2) for Canadian permanent residence via Express Entry French-language draws.

## What's inside

- **`index.html`** — Interactive React dashboard with week navigation, daily schedules, grammar tables, writing prompts, speaking tasks, habit tracker, and progress tracking.
- **`data.js`** — 26 weekly data objects covering every day from March 16 to September 16, 2026. Each day has 6–9 time slots mapped to your real routine (dog walk, commute, coffee break, lunch, evening).

## How to use

Open `index.html` in any browser. Both files must be in the same folder. No build step, no dependencies, no server needed.

## Plan structure

| Month | Weeks | CEFR | Focus |
|-------|-------|------|-------|
| 1 | 1–4 | A1 | Articles, pronouns, present tense, descriptions, reflexive verbs |
| 2 | 5–8 | A2 | Passé composé, imparfait, object pronouns, future tense |
| 3 | 9–12 | B1 | PC vs imparfait, conditional, relative pronouns, subjunctive intro |
| 4 | 13–16 | B1+/B2 | Subjunctive deepening, advanced pronouns, connectors, conditional past |
| 5 | 17–20 | B2 | TEF exam prep — listening, reading, writing, speaking intensives |
| 6 | 21–26 | B2 | Weakness repair, full simulations, exam day, IRCC submission |

## Daily routine integration

Every weekday follows this structure:

- **7:00 AM** — Dog walk + French podcast (CBF / InnerFrench)
- **8:00 AM** — Commute → work + audio (30 min)
- **9:00 AM** — Coffee break + Duolingo/vocab (10 min)
- **12:30 PM** — Lunch: grammar (Kwiziq) + writing + reading (30 min)
- **5:00 PM** — Anki review (10 min, non-negotiable)
- **5:15 PM** — Commute ← home + audio replay (30 min)
- **8:00 PM** — Evening: speaking practice + journaling (20 min)

## Key resources

- [Kwiziq French](https://french.kwiziq.com/) — Grammar lessons with specific URLs per topic
- [Coffee Break French](https://youtube.com/coffeebreaklanguages) — Podcast (YouTube)
- [InnerFrench](https://innerfrench.com/podcast/) — Intermediate podcast
- [Anki](https://apps.ankiweb.net/) — Spaced repetition (20–25 new cards/day)
- [TV5Monde](https://apprendre.tv5monde.com/) — Reading + listening exercises by CEFR level
- [BonPatron](https://bonpatron.com/) — French grammar checker for writing
